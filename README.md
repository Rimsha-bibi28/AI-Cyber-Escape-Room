# AI Cyber Escape Room

## Project Overview

The AI Cyber Escape Room is a gamified web-based platform designed to enhance cybersecurity awareness through interactive learning. Users solve real-world cybersecurity challenges to escape a virtual cyber lab, making the learning process engaging and practical.

## Key Features

- Phishing URL detection using machine learning
- Website phishing analysis based on security features
- Password strength validation using AI models
- Gamified experience with levels, score system, and timer
- Web-based interface built with Flask, HTML, CSS, and JavaScript

## Technologies Used

- Python
- Machine Learning (Scikit-learn)
- Flask (Web Framework)
- HTML, CSS, JavaScript
- Kaggle Datasets for model training
